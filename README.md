# Expense Tracker

## Introduction
Help users record, view, modify and delete their daily expenses, and support management by title, category, amount, date and description. 

## Technology Stack
- **Frontend**: React + Vite + Axios
- **Backend**: FastAPI (Python)
- **Database**: MySQL + SQLModel (ORM)
- **Vision Control**: Git + GitHub

## Function List
- Add expenses (Create)
- View all expense list (Read)
- Edit expenses (Update)
- Delete expenses (Delete)
- Single-page application experience, no need to refresh the page
- Responsive design
